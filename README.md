# C++ Practice

Learn basic syntax for c++

## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/Sleepy4k/cpp-practice.git
~~~

Go to the project directory  

~~~bash  
  cd cpp-practice
~~~

Start the cpp file  

~~~bash  
  g++ <filename> -o <output name>
~~~

## Usage/Examples

~~~bash
  g++ hello.cpp -o hai
~~~  

~~~bash
  ./hai
~~~
